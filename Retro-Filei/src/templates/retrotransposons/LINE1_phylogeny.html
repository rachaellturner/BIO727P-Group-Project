LINE-1 Phylogeny


The evolutionary relationship between sequences of LINE-1 repeat families can be seen here. The evolutionary relationship is shown as a phylogenetic tree and a circular cladogram.

The cladogram shows the different clades of the repeats that generate different open reading frames. The branch length in a phylogenetic tree represents the evolutionary history, therefore the phylogenetic tree allows the genetic distance between the different repeats to be seen.

These dendrograms use a consensus sequence for each repeat generated by multiple sequence alignments.
